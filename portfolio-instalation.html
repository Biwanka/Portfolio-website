Installation

This is an installation that contains 13 different works. The materials range from, wood, fabric, carpet, silk fabric, canvas and paper. Each work used coloured pencils, ink, paint pens, alcohol pens, pens and acrylic paints to make it. I use a lot of my personal space and surroundings as inspiration for my work. This work uses my body measurements. All of the artworks are drawn from my point of view in my own home. Bringing the viewer into this installation where these small scattered places can create a complete environment. This installation uses my height and perspective. This room brings the viewers to enter my world, as if in the middle of the night and try to place themselves in my place. The work uses my body parts which are drawn and displayed from my body scale as a way to help viewers enter my world.
